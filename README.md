# My-Portfolio-Client

## Installation:
1. Clone the repository.
2. Install dependencies using `npm install`.
3. Create `.env` file to root and assign appropriate value.

    ```bash
    USER_EMAIL=
    USER_PASSWORD=
    ```
4. Run the server using `npm run dev`.

### Endpoints:
- /dashboard
- /dashboard/add-skill
- /dashboard/add-project
- /dashboard/add-blog
